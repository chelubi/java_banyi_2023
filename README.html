Опис Структури:

 • src/: Основна директорія з кодом.

 • main/: Всі основні файли програми.

 • java/: Java-класи програми.

 • quizmaker/: Основні класи вашої вікторини.

 • utils/: Утиліти, такі як зчитувачі та записувачі файлів.

 • resources/: Непрограмні ресурси.

 • images/: Зображення, використані в проекті.

 • data/: Дані для вікторин, такі як CSV та JSON файли.

 • test/: Тестові класи.

 • java/: Java-класи для тестування.

 • quizmaker/: Тести для класів вікторин.

 • utils/: Тести для утиліт.

 • resources/: Ресурси для тестування.

 • testData/: Тестові файли, такі як NZ_test.dat та NZ_test.txt.
